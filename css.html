CSS Dateien

./css/font-awesome.min.css

CSS Datei für die Icon Schrifart Font-Awesome

Bitte nicht ändern, da die Datei nur die Icons als CSS class zur Verfügung stellt!

./css/Template Stil/bootstrap.min.css

Die CSS Framework Datei von Bootstrap und damit auch die wichtigste :).

Veranwortlich für das Layout, Schrift, Größe und Farben, ein verändern der Datei selber ist wegen zukünftigen Updates von Bootstrap nicht zu empfehlen, um dennoch das Gästebuch an die Webseite anpassen zu können gibt es 2 Möglichkeiten.

  1. Eine persönliche Datei unter Bootstrap Customize erstellen.
  2. Das Überschreiben der CSS Anweisungen (bootstrap-theme), mehr dazu unter eigenen Stil erstellen, anpassen.

./css/Template Stil/bootstrap-theme.css

Dient für persönliche Änderungen der bootstrap.min.css ohne die eigentliche Datei verändern zu müssen.

./css/Template Stil/style.css

Enthält nur spezielle für das Gästebuch notwendige CSS Anweisungen

Eigenen Stil erstellen, anpassen

  1. Erstelle einen neuen Ordner im Verzeichniss ./css/, der Name des Ordners wird später in der Administration als Stil angezeigt.
  2. Lade von Bootstrap die aktuelle Version oder stelle dir unter Bootstrap Customize eine persönliche Datei zusammen, enpacke die CSS Dateien in den eben erstellten Ordner.
